|*Requirement 2* {set:cellbgcolor:#CACCCE}|Req/table-defs/tile-ancillary +
 +

A GeoPackage that contains tiled gridded data SHALL contain a `gpkg_2d_gridded_tile_ancillary` table or view as per http://www.geopackage.org/spec/#gpkg_2d_gridded_tile_ancillary_table[_Tile Ancillary Table Definition_]. Subsequent extensions or custom implementations MAY add additional columns to this table. Clients SHALL ignore additional columns that are unrecognized.{set:cellbgcolor:#FFFFFF}
