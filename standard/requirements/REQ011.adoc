|*Requirement 11* {set:cellbgcolor:#CACCCE}| Req/gpkg-2d-gridded-tile-ancillary-values +
 +

Values of the `gpkg_2d_gridded_tile_ancillary` `tpudt_name` column SHALL reference existing http://www.geopackage.org/spec/#tiles_user_tables[_tile pyramid user data tables_] and rows in `gpkg_2d_gridded_coverage_ancillary`. When the datatype of the corresponding `gpkg_2d_gridded_coverage_ancillary` row is _float_, the `scale` and `offset` values SHALL be set to the defaults. {set:cellbgcolor:#FFFFFF}
